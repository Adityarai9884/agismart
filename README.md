# 🌿 AgriSmart — AI-Powered Digital Mandi System

## 📌 Overview

AgriSmart is a digital mandi platform that helps farmers, wholesalers, and market incharge users interact efficiently. It provides price insights, transport calculations, and AI-powered assistance.

---

## 📁 Project Structure

```
agrismart/
├── frontend/               
├── backend/                
└── docs/
```

---

## 🚀 Quick Start

### Frontend

```
cd frontend
npm install
npm start
```

---

## 🛠 Tech Stack

| Layer    | Technology           |
| -------- | -------------------- |
| Frontend | React.js             |
| Backend  | Node.js, Express.js  |
| Database | MongoDB              |
| Auth     | JWT                  |
| AI       | Gemini API (planned) |

---

## ✨ Features

* Farmer dashboard
* Wholesaler dashboard
* Transport cost calculator
* Price ticker
* AI chat (planned)
